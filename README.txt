# Ferroamp Control for Homey

Control your Ferroamp EnergyHub system directly from Homey using Flow cards.

## Features
- Charge or discharge the battery at a specified power level
- Set Self-Consumption mode
- Set Peak Shaving mode
- Limit solar export to the grid
- Limit grid import for EV charging and PowerShare
- Full configuration of all Ferroamp EnergyHub settings

## Setup
1. Add the Ferroamp EnergyHub device
2. Enter your System ID (found in the Ferroamp Portal URL: ?id=XXXX)
3. Enter your Ferroamp Portal email and password
4. Start automating with Flow cards

## Requirements
- A Ferroamp EnergyHub system
- A Ferroamp Portal account (portal.ferroamp.com)
